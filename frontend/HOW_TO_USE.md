# 如何使用文本归档助手

## 访问您的数据

### 1. 查看备忘录
- 登录后，点击顶部导航栏的 **📝 备忘录** 链接
- 或访问：`http://localhost:5173/memos`
- 在这里您可以：
  - 创建新备忘录
  - 查看所有备忘录（按时间倒序）
  - 编辑现有备忘录
  - 删除备忘录
  - 按标签筛选备忘录

### 2. 查看日程
- 登录后，点击顶部导航栏的 **📅 日程** 链接
- 或访问：`http://localhost:5173/schedules`
- 在这里您可以：
  - 创建新日程
  - 查看所有日程（按时间排序）
  - 编辑现有日程
  - 删除日程
  - 按日期范围筛选日程

### 3. OCR识别
- 登录后，点击顶部导航栏的 **🔍 OCR识别** 链接
- 或访问：`http://localhost:5173/ocr`
- 上传图片进行文字识别

## 导航栏功能

顶部导航栏提供快速访问：
- **文本归档助手** - 返回首页
- **📝 备忘录** - 查看和管理备忘录
- **📅 日程** - 查看和管理日程
- **🔍 OCR识别** - 图片文字识别
- **🚪 登出** - 退出登录

## 数据存储位置

所有数据都存储在后端数据库中：
- 数据库文件：`backend/text_archive.db`
- 每个用户的数据完全隔离
- 支持跨设备同步（同一账号在不同设备登录可看到相同数据）

## 启动应用

### 启动后端
```bash
cd backend
conda activate ocr_agent
uvicorn main:app --reload
```

### 启动前端
```bash
cd frontend
npm run dev
```

然后访问：http://localhost:5173

## 常见问题

### Q: 为什么看不到我的数据？
A: 请确保：
1. 已经登录
2. 后端服务正在运行
3. 检查浏览器控制台是否有错误

### Q: 如何添加标签？
A: 在备忘录创建/编辑表单中，输入标签并用逗号分隔，例如：`工作, 学习, 想法`

### Q: 如何筛选数据？
A: 
- 备忘录：在标签筛选框中输入标签（用逗号分隔），点击"筛选"
- 日程：选择开始日期和结束日期，点击"筛选"

### Q: 数据会丢失吗？
A: 不会。所有数据都持久化存储在数据库中，除非您手动删除。
